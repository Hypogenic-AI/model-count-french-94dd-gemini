# Cloned Code Repositories

## Repo 1: Multilingual Number Puzzles
- **URL**: [https://github.com/antara-raaghavi/multilingual-number-puzzles](https://github.com/antara-raaghavi/multilingual-number-puzzles)
- **Purpose**: Official implementation of Bhattacharya et al. (2025) "Investigating the interaction of linguistic and mathematical reasoning in language models using multilingual number puzzles".
- **Location**: `code/multilingual-number-puzzles/`
- **Key Files**:
    - `code/mathify_new.py`: Main logic for generating explicit/implicit puzzles.
    - `code/query_o1.py`: Example of querying OpenAI models with these puzzles.

## Application to French Research
The `mathify` logic can be adapted to generate "mathified" French number puzzles. For example, converting "quatre-vingt-onze" into explicit operations like `4 * 20 + 11` to test if the model's reasoning improves.
